# Bank Account Manager

A simple Python console project to manage a bank account. This application allows users to deposit and withdraw money, check their current account balance, and view transaction history. Designed for beginners to practice basic programming concepts and file handling in Python.

## Features

- Deposit money to your account.
- Withdraw money with balance checking.
- View your account balance.
- See transaction history (deposits and withdrawals).
- Easy-to-use menu system.

## Installation

1. Download or clone this repository:
  git clone https://github.com/Surya6507/Suryakarnik22-pie.git
2. Make sure you have Python 3 installed on your system.

## Usage

Run the program using:
python vityarthi_project.py
Enter the menu options as prompted to deposit, withdraw, check balance, and view transactions.

## Input and Output

- Enter deposit and withdrawal amounts as positive numbers.
- All actions and results will be displayed in the console.
- Transaction history shows all previous deposits and withdrawals.
