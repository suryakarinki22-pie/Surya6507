# Problem Statement
To help users manage their bank accounts with basic operations like deposit, withdrawal, and balance check using Python.

# Scope of the Project
The project provides a console-based bank account manager. It supports basic transactions: deposit, withdrawal, balance enquiry, and transaction history.

# Target Users
Python beginners, students, or anyone wanting to learn basic programming concepts through a simple simulation.

# High-Level Features
- Deposit money
- Withdraw money
- Check account balance
- View transaction history
- Simple menu navigation
